# Python Practice Notebook

This repository contains a Jupyter Notebook named **Python_Practice.ipynb**, which includes Python practice exercises, examples, and explanations designed to improve your programming skills. It is suitable for beginners, students, and anyone looking to strengthen their Python fundamentals.

---

##  Project File

**Python_Practice.ipynb**  
A Jupyter Notebook containing multiple Python exercises such as basic syntax, loops, functions, data structures, and beginner-level programs.

This notebook helps you learn Python with simple, clear examples.  
It includes:

- Basic Python commands  
- Variables and data types  
- Conditional statements  
- Loops (for, while)

- File operations  
- Basic problem-solving programs
